# imgui_test

Test for creating a calibration application using ImGui
## Getting Started
Follow steps   [here.](https://blog.kortlepel.com/c++/tutorials/2023/03/16/sdl2-imgui-cmake-vcpkg.html).

## Building
To setup cmake, run:
```bash
cmake -S . -B bin
```

To build the application:
```bash
cmake --build bin
```

To run the application:
```bash
./bin/my-test 
```
